# AR-Anatomy-Explorer
AR Anatomy Explorer: Mobile-Based Spatial Learning Prototype
## Description
A mobile augmented reality (AR) prototype developed using Unity AR Foundation that allows users to place and interact with 3D objects in real-world environments. The system supports spatial learning through embodied interaction and contextual visualization.
## Features
* Real-time plane detection (ARCore)
* Tap-to-place 3D object in physical space
* Interactive object repositioning
* Contextual information display (tap interaction)
* Reset and repeat learning cycle
## Research Value
This prototype explores spatial cognition and embodied learning using AR interfaces. It contributes to research in immersive learning environments and interaction design in extended reality systems.
## Tools & Technologies
* Unity
* Unity AR Foundation
* ARCore (Android)
* Blender (3D assets)
